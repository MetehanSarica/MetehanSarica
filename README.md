## 🚀 About Me

I'm a software developer and a second-year Computer Programming student at Afyonkarahisar University. I'm passionate about building clean, scalable, and intelligent applications. My core focus spans enterprise-level desktop applications, backend architecture, and seamlessly embedding AI into local and cloud workflows.

I’m currently working on:

- **Saku Kaze** – Blazing-fast, multi-language offline desktop text editor (Tauri, Rust, Svelte)
- **Sentinel** – Commercial-grade stock automation SaaS (.NET WPF, DevExpress, Dapr)
- **S.A.K.U.** – Local voice-activated AI system assistant (Python, Ollama)
- **Notoid** – AI-powered study assistant (C#, .NET, DevExpress & Gemini API)
- **Zero Friction** – Fast-paced, movement-based multiplayer FPS (Unity)
- **metehansarica.dev** – My portfolio website built with Next.js + TailwindCSS

## 📦 Featured Project

### 🔹 **Saku Kaze**
A blazing-fast, multi-language offline desktop text editor engineered for extreme performance. Built from the ground up to replace heavy, resource-intensive Electron-based editors, Saku Kaze leverages Rust for a highly secure, low-memory backend and Svelte for a fluid, reactive user interface. It is designed to handle large codebases effortlessly, providing developers with a sleek, native-like experience without the bloat.

Tech Stack: **Tauri, Rust, Svelte**
